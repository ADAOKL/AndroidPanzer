## 🆘 HÄUFIGE PROBLEME

### Problem 1: ADB nicht erkannt
```
Lösung:
  1. USB-Kabel überprüfen
  2. USB-Debugging prüfen (Settings → Developer)
  3. Gerät autorisieren (Popup bestätigen)
  4. ADB rekalibrieren: adb kill-server && adb start-server
```

### Problem 2: Menü unlesbar
```
Lösung:
  1. Terminal-Größe ändern (größer)
  2. Schrift kleiner machen (Terminal-Settings)
  3. Dark Mode aktivieren (besserer Kontrast)
  4. LANG=de_DE.UTF-8 setzen
```

### Problem 3: Tool bricht ab
```
Lösung:
  1. Gerät überprüfen (noch verbunden?)
  2. Log-Datei lesen: cat ~/.apz/log.txt
  3. ADB-Fehler prüfen: adb shell
  4. GitHub Issue öffnen mit Log
```

### Problem 4: Keine Berechtigungen
```
Lösung:
  1. Root-Zugriff geben
  2. Su-Binär prüfen: adb shell su -c "id"
  3. SELinux deaktivieren (falls möglich)
  4. Fallback-Methode verwenden
```
